V5 is the version currently live on the production site; V1–V4 (+ V3a–V3d) are alternates — swap one of them into project.html's case 'slider' if you want a different feel. V3 — Auto-advance with dwell

Autoplay2s dwell1.1s easeDotsHover pauses

Each slide holds for 2 seconds, then eases smoothly to the next over ~1.1s. Dots show position and let you jump. Hover anywhere to pause. Loops back from the last slide to the first.

V3b — Ken Burns auto

Autoplay6.5s dwellCross-fadeSlow zoom + driftDots

Each slide slowly zooms and drifts (1.0× → 1.15×, slight pan) while it's on screen, then cross-fades to the next. Cinematic. Best when you want imagery to feel alive even when standing still — long dwell so the motion is visible.

V3d — Stepped marquee

AutoplayStepped (no smooth scroll)No hover pausePill indicatorsLoops seamlessly

Steps frame-by-frame with a brief settling ease — tick · tick · tick — like a flipboard or stepping motor. Never pauses, even on hover. Pill-shaped indicators below mark the current frame (active pill widens). Frames are doubled in the DOM so the wraparound is invisible.

V5 — Production (live on project pages)

Live rendererScroll-snapCustom RAF tweenAutoplay opt-in autoplay = true

What project.html actually renders for a slider token today. Full-bleed scroll-snap (no peek, no chrome), pagination dots only. Custom requestAnimationFrame tween (ease-in-out cubic, 900ms) replaces native smooth-scroll so timing stays consistent across devices. Autoplay is opt-in via the token's autoplay=true option — this demo has it on so you can see the 2.5s cycle. Pointer/wheel cancels the in-flight tween; clicking a dot stops auto permanently; pauses when scrolled off-screen; honors prefers-reduced-motion.

Quick guide

V1 Classic for most cases — packaging variants, before/after, brand applications.
V2 Cross-fade for moodboards / loosely related frames.
V3 Marquee for ambient detail shots — patterns, textures, repetitions.
V4 Snap row for catalog feel — show me everything, let me pick.
V5 Production — currently live. Full-bleed scroll-snap with optional auto-cycle. Switching the live renderer to V1/V2/V3/V4 means swapping the case 'slider' block in project.html with that variant's HTML + JS.
